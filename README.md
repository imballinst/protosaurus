# protosaurus

To generate docusaurus documentation website from your `*.proto`.

## Quick start

You need to have [Go 1.17.x](https://go.dev/doc/install) installed on your system, also: `bash`, `make`, `awk`, `tar`, `unzip`, and `curl`.

To generate:

```console
make gen
```

To check on more targets:

```console
make help
```
